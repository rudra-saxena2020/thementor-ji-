Implementation Details

The authentication system uses JWT tokens stored in localStorage and communicates with the backend at https://three-0-4jps.onrender.com.

Key Components:

  • AuthService - Centralized service for authentication API calls
  • AuthContext - Manages authentication state throughout the app
  • UserAuthInfo - Displays auth info in the Profile page
  • AuthTestPage - Dedicated page for testing authentication features
